# Procomic Extension

امتداد Tachiyomi لموقع Procomic العربي

## الميزات

- تصفح أحدث المانجا والكوميكس
- البحث عن المانجا
- قراءة الفصول
- تخزين محلي للتفضيلات
- إمكانية تغيير رابط الموقع
- إخفاء الفصول المدفوعة
- وضع التصفح الآمن

## الإعدادات

- **تغيير رابط الموقع**: غير الرابط الافتراضي للموقع
- **إخفاء الفصول المدفوعة**: أخف الفصول التي تحتاج اشتراك
- **التصفح الآمن**: فعّل وضع التصفح الآمن

## التثبيت

1. انسخ ملفات الامتداد إلى مجلد Tachiyomi
2. أعد تشغيل التطبيق
3. فعّل الامتداد من إعدادات المصادر
